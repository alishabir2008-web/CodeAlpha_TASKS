# Language Translation Tool - CodeAlpha AI Internship Task 1

A responsive web-based language translation application that leverages AI for accurate text translation between multiple languages.

# Features

- **Multi-language Support**: Translate between 12+ languages including English, Spanish, French, German, and more
- **Auto-detection**: Automatically detect source language
- **Text-to-Speech**: Listen to both source and translated text
- **Swap Languages**: Quickly switch between source and target languages
- **Copy to Clipboard**: Easy copying of translated text
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time Character Count**: Track input and output text length
- **Clean UI**: Modern, intuitive interface with smooth animations

This project is part of Task 4 of the CodeAlpha Artificial Intelligence Internship.
The objective of this task is to implement real-time object detection and tracking using computer vision and AI techniques.

The system detects objects from a live webcam feed and assigns unique tracking IDs to each detected object, allowing them to be tracked consistently across video frames.

Draw bounding boxes and labels on detected objects

Track objects across frames using the SORT tracking algorithm

Display detection and tracking results in real time

The webcam captures real-time video frames

Each frame is passed to the YOLO model for object detection

Detected objects are represented with bounding boxes

SORT assigns a unique ID to each object

Objects are tracked smoothly across consecutive frames

The final output is displayed with labels and tracking IDs
