# Language Translation Tool - CodeAlpha AI Internship Task 1

A responsive web-based language translation application that leverages AI for accurate text translation between multiple languages.

# Features

- **Multi-language Support**: Translate between 12+ languages including English, Spanish, French, German, and more
- **Auto-detection**: Automatically detect source language
- **Text-to-Speech**: Listen to both source and translated text
- **Swap Languages**: Quickly switch between source and target languages
- **Copy to Clipboard**: Easy copying of translated text
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time Character Count**: Track input and output text length
- **Clean UI**: Modern, intuitive interface with smooth animations
